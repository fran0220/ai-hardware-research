# AI硬件市场调研报告 2025

> 17个AI硬件品类深度市场调研与投资机会分析

## 📊 项目简介

本项目对2025年AI硬件市场进行全方位调研,涵盖17个品类,包含8份详细品类报告、40+产品深度分析、5大失败案例剖析。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run docs:dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run docs:build
```

构建产物位于 `docs/.vitepress/dist/`

### 预览构建结果

```bash
npm run docs:preview
```
